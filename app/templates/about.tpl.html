About PromptLy

PromptLy is a web app created to help writers write with various prompts.

A typed page averages to be 250 words, so we created a writing window that will hold a maximum of 250 words, so writers can write until they have a full page.

Coming soon we will have a timer feature that allows writers to set a countown to write as quickly as they can until they fill the page or run out of time.
